
# گزارش پروژه پایانی آزمایشگاه سیستم عامل

## نام پروژه
**User Activity Reporter** - ابزار گزارشگر فعالیت کاربران

## ۱. مقدمه
این پروژه به عنوان یک ابزار خط فرمان (CLI) برای پایش فعالیت کاربران در سیستم‌های لینوکس طراحی شده است. این ابزار به سیستم‌مدیران امکان می‌دهد تا فعالیت کاربران را به صورت زنده و با جزئیات کامل بررسی کنند. این پروژه از لیست ۲۱ گزینه پیشنهادی انتخاب شده و با توجه به نیازهای عملیاتی سیستم‌های لینوکس، انتخاب شده است.

## ۲. اهداف پروژه
- ایجاد یک ابزار خط فرمان برای پایش فعالیت کاربران
- ارائه گزارش‌های جامع با فرمت‌های مختلف (جدول، JSON، CSV)
- پشتیبانی از حالت پایش زنده (Real-time Monitoring)
- امکان فیلتر و مرتب‌سازی بر اساس معیارهای مختلف
- ایجاد بسته‌بندی استاندارد برای نصب و استفاده

## ۳. ویژگی‌های اصلی
### ۳.۱. گزارش‌های فعالیت کاربران
- نمایش زمان آخر ورود کاربر
- محاسبه مدت زمان آنلاین بودن کاربر
- شمارش فرآیندهای فعال هر کاربر
- نمایش رنگی برای کاربران با فرآیندهای بیش از حد

### ۳.۲. حالت پایش زنده
- امکان نمایش زنده فعالیت کاربران با تنظیم فاصله زمانی
- امکان متوقف کردن پایش با Ctrl+C

### ۳.۳. فرمت‌های خروجی
- جدول خوانا (پیش‌فرض)
- فرمت JSON برای پردازش خودکار
- فرمت CSV برای وارد کردن به اکسل

### ۳.۴. تنظیمات قابل پیکربندی
- تنظیم فرمت خروجی پیش‌فرض
- تنظیم محدوده فرآیندهای هشدار
- تنظیم زمان پایش زنده

## ۴. ساختار پروژه

```
│   └── user-activity.conf        # فایل پیکربندی (غیر اجرایی)
├── man/                          # صفحات راهنما
│   ├── user-activity-reporter.1  # man برای user-activity-reporter

│   ├── control                   # اطلاعات بسته
│   ├── changelog                 # تاریخچه تغییرات
│   ├── rules                     # قوانین ساخت
│   ├── copyright                 # مجوز
│   ├── install                   # فهرست نصب
│   └── source/format

│   └── user-activity-reporter.spec # فایل spec

```

## ۵. نحوه نصب و استفاده
### ۵.۱. نصب
#### برای سیستم‌های دبیان/اوبونتو:
```bash
sudo apt install build-essential debhelper devscripts
./scripts/build-deb.sh
sudo dpkg -i build/deb/user-activity-reporter_1.0.1_all.deb
```

#### برای سیستم‌های RPM (فدورا/رزل):
```bash
sudo dnf install rpm-build rpmdevtools
./scripts/build-rpm.sh
sudo rpm -i build/rpm/user-activity-reporter-1.0.1.noarch.rpm
```

### ۵.۲. استفاده
#### نمایش فعالیت کاربران:
```bash
user-activity-reporter
```

#### نمایش فعالیت کاربر خاص:
```bash
user-activity-reporter -u fargol
```

#### پایش زنده با فاصله ۵ ثانیه:
```bash
user-activity-reporter -w
```

#### خروجی به فرمت JSON:
```bash
user-activity-reporter -o report.json -f json
```
